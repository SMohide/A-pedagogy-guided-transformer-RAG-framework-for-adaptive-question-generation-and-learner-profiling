EduGen — Adaptive AI-Based Question Generation System

A research-grade Transformer–RAG based adaptive assessment framework designed to generate personalized, pedagogy-aware quiz questions.

Key Features:

Developed a multi-model learner profiling system combining:
Performance Probability Estimator (PPE)
Pedagogical Rule Extraction Model (PREM)
Learner Behaviour Segmentation (LBSM)
Latent Learning Pattern Extractor (LLPE)
Implemented Item Response Theory (IRT) for:
Learner ability estimation
Difficulty calibration of questions
Designed three core algorithms:
APDEA – Adaptive profiling and difficulty estimation
CTQS – Context-aware Transformer-RAG question generation
BSSCA – Benchmark and scholarship suitability computation
Integrated Bloom’s Taxonomy for cognitive-level question generation (Remember, Apply, Analyze)
Built a RAG pipeline to ensure:
Factually correct outputs
Reduced hallucination in LLM responses
Achieved strong performance:
Accuracy up to 91.2%
F1-score up to 89.4%, outperforming GPT-4 baselines
Supported both:
High-performance LLM-based system
Lightweight SLM-based scalable version for low-resource environments
Generated benchmark scores and scholarship suitability metrics for learners using data-driven evaluation

This system moves beyond simple question generation and works as a complete adaptive learning and evaluation ecosystem.
